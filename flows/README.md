# Flow Docs

These files are short control-flow maps for the live supervisor semantics.

- `theorem_stating.md`: broad theorem-stating, invalid retries, reviewer-controlled reset
- `proof_formalization.md`: proof worker outcomes, verification, reviewer control
- `cleanup.md`: proof-complete style cleanup
- `verification_gating.md`: correspondence and soundness gating rules
- `reset_and_rewind.md`: exact committed resets and rewinds

These are intended to stay close to the code, not to replace it.
